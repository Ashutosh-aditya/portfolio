Analog Clock Application

This analog clock application is designed using C++ and OpenGL (GLUT) to display an interactive analog clock.


It features:

Clock Display:

  • The clock features three hands: hour, minute, and second, which rotate to show the current time.
  • The clock face is divided into 12 segments, with either Roman numerals or digits for hour representation, depending on user preference.

Interactive Options:

  • Users can choose the color of the clock hands from a context menu (Orange, Green, or Blue).
  • A context menu also allows users to toggle between Roman numerals and digits for hour representation.
